# JotSpace 📝

A modern and responsive note-taking application built with React for capturing ideas, tasks, and thoughts.

## Overview

**JotSpace** is a clean and minimal note-taking application designed to help users quickly create, organize, and manage notes in one place.

Built with a focus on simplicity and performance, JotSpace provides a smooth experience for capturing ideas, reminders, and daily thoughts.

## Features 

- Create, edit, and delete notes 
- Pin and unpin notes 
- Mark notes as important 
- Archive and unarchive notes 
- Search notes instantly 
- Persistent storage using LocalStorage 
- Clean and minimal UI 
- Toast notifications for actions

## Tech Stack

- React.js
- JavaScript (ES6+)
- CSS / Tailwind CSS
- React Hooks
- LocalStorage

## Setup

### Clone the Repository

```bash
git clone https://github.com/NandithaB19/JotSpace.git
```

### Navigate to the Project

```bash
cd JotSpace
```

### Install Dependencies

```bash
npm install react-hot-toast
npm install react-icons
npm install react-router-dom
npm install tailwindcss
```

### Start Development Server

```bash
npm run dev
```
