# JotSpace 📝

A modern and responsive note-taking application built with React for capturing ideas, tasks, and thoughts.

## Overview

**JotSpace** is a clean and minimal note-taking application designed to help users quickly create, organize, and manage notes in one place.

Built with a focus on simplicity and performance, JotSpace provides a smooth experience for capturing ideas, reminders, and daily thoughts.

## Tech Stack

- React.js
- JavaScript (ES6+)
- CSS / Tailwind CSS
- React Hooks
- LocalStorage

## Setup

### Clone the Repository

```bash
git clone https://github.com/your-username/JotSpace.git
```

### Navigate to the Project

```bash
cd JotSpace
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

The application will run locally at:

```bash
http://localhost:5173
```
